Get JonahStrap

Get started by learning how to install JonahStrap and how you should set up your HTML to get it working!

Install JonahStrap

Method 1 (recommended):

Click here to download the JonahStrap CSS file.

Import it to your website, then add the following code into your <head> tag. This method is recommended because it lets you edit the CSS file to change the colors and other styles. You can quickly change the colors of the CSS at the very top of the jonahstrap.css file in Variables (line 6).

<link rel="stylesheet" type="text/css" href="jonahstrap.css">


Method 2:
Copy and paste this code into your <head> tag.

<link rel="stylesheet" type="text/css" href="https://css.jonahkc.com/jonahstrap.css">

Set Up Your HTML

This is my recommended <head> tag. Bolded things are required for JonahStrap to work, everything else is fancy meta tags for things like description and favicon.

<head>
    <title>PAGE TITLE</title> <!-- Page Title -->
    <link rel="icon" href="image.png"> <!-- Favicon -->
    <meta name="description" content="PAGE DESCRITION"> <!-- Page Description -->
    <meta name="keywords" content="PAGE KEYWORDS"> <!-- Keywords -->
    <meta name="robots" content="index, follow"> <!-- Index Preferences -->
    <link rel="canonical" href="https://example.com"> <!-- Canonical Domain -->
    <meta charset="UTF-8"> <!-- Unicode Type -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- Screen Width -->
    <link rel="stylesheet" type="text/css" href="jonahstrap.css" media="screen"> <!-- Use jonahstrap.css file (replace this line if you use method #2-->

    <!-- Social Media Embeds -->
    <meta property="og:type" content="website">
    <meta property="og:url" content="https://example.com/"> <!-- Canonical Domain -->
    <meta property="og:title" content="PAGE TITLE"> <!-- Page Title -->
    <meta property="og:description" content="PAGE DESCRITION"> <!-- Page Description -->
    <meta property="og:image" content="https://example.com/image.png"> <!-- Social Media Embed (ideally 16x9 ratio, and must be link) -->

    <!-- Special Twitter Embeds -->
    <meta property="twitter:card" content="summary_large_image">
    <meta property="twitter:url" content="https://example.com/"> <!-- Canonical Domain -->
    <meta property="twitter:title" content="PAGE TITLE"> <!-- Page Title -->
    <meta property="twitter:description" content="PAGE DESCRITION"> <!-- Page Description -->
    <meta property="twitter:image" content="https://example.com/image.png"> <!-- Social Media Embed (ideally 16x9 ratio, and must be link) -->
</head>
