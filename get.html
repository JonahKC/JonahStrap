Get JonahStrap

Get started by learning how to install JonahStrap and how you should set up your HTML to get it working!

Install JonahStrap

Method 1 (recommended):
Click here to download the JonahStrap CSS file. Import it to your website, then add the following code into your <head> tag. This method is recommended because it lets you edit the CSS file to change the colors and other styles.

<link rel="stylesheet" type="text/css" href="jonahstrap.css" media="screen">


Method 2:
Copy and paste this code into your <head> tag.

<link rel="stylesheet" type="text/css" href="https://css.jonahkc.com/jonahstrap.css">

Set Up Your HTML

This is my recommended <head> tag. Bolded things are required for JonahStrap to work, everything else is fancy meta tags for things like description and favicon.

<head>
    <title>PAGE TITLE</title>
    <meta name="description" content="PAGE DESCRITION">
    <link rel="icon" href="image.png">
    <meta property="og:image" content="https://example.com/image.png">
    <meta name="keywords" content="PAGE KEYWORDS">
    <meta name="robots" content="index, follow">
    <link rel="canonical" href="https://example.com">
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" type="text/css" href="jonahstrap.css" media="screen" />
</head>
