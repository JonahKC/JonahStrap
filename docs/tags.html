Tags

Here's all the basic CSS used on the HTML tags (like <a>) by default! If you're looking for them in the CSS file, they start at line 17.

<html>

Sets font, margins, background, and text color.

<a>

Underlines and makes links the same color as normal text.

<img>

Makes images undraggable and unselectable

<footer>

Adds wider margins, then centers and shrinks text.

<button>

Colors buttons and gives them padding and margins.

<input> and variations

Gives inputs backgrounds and borders and gives them padding and margins.

<table>, <th>, <td>

Styles your tables with some very snazzy borders and backgrounds.
