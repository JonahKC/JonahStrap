Title Bar

Here's all CSS for setting up a top bar like the one on this site. You can change the title font independently by changing the variable altfont. If you're looking for the classes in the CSS file, they start at line 353.

.topbar, .toptitle, and .topbutton

.toptitle and .topbutton must go inside .topbar for the bar to work and the CSS to not spread to the rest of the page. .toptitle puts your page title in the top left corner, taking up 50% of screen width. .topbutton puts your links or buttons opposite to your title. Your HTML should look something like this:

<div class="topbar">
    <div class="toptitle">
        <h1><a class="nodeco" href="https://example.com"><img src="image.png" alt="Logo" width="30" height="30">Name</a></h1>
    </div>
    <div class="topbutton">
        <h2><a class="nodeco" href="link">Your Link</a></h2>
    </div>
</div>

.dropdown, .dropbtn, .dropdown-content

Everything must go inside .dropdown for the dropdown to work. .dropbtn is the button you hover over. .dropdown-content is what the links go in. The HTML should look something like this:

<div class="dropdown">
    <button class="dropbtn">Dropdown</button>
    <div class="dropdown-content">
        <a href="Link">Name</a>
        <a href="Link">Name</a>
        <a href="Link">Name</a>
    </div>
</div>
